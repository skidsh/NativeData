# NativeData.Sqlite

SQLite provider package for NativeData.

## Contents

- `SqliteConnectionFactory`
- `SqliteSqlDialect`
- Integration with `Microsoft.Data.Sqlite`

## Build

```bash
dotnet build src/NativeData.Sqlite/NativeData.Sqlite.csproj
```

## Packaging

This project is packable and published as a NuGet package.
